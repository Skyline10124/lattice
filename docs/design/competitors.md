# 竞品分析

| 项目 | 类型 | vs LATTICE |
|------|------|-----------|
| OpenRouter | 模型路由 SaaS | LATTICE 是库，不绑 SaaS |
| LiteLLM | 模型网关 | LATTICE Rust 原生，更快 |
| LangGraph | 多 agent 框架 | 重 vs 轻 |
| CrewAI | 多 agent 编排 | 重框架 vs 轻库 |
| n8n / Dify | 可视化工作流 | 不同赛道 |
| Claude Code | 编码 agent | LATTICE 底层引擎，不绑一家模型 |
| MCP | 模型→工具协议 | 互补，插件内部引用 |
| A2A | Agent 通信协议 | handoff 层参考 |
